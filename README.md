# QuizApp
Projeto realizado em React e JavaScript
1. Baixe o arquivo
2. Abra o CMD dentro da pasta app-quiz
3. Inicie o servidor com "npm run dev"
4. Abra o narvegador com o link do servidor
5. Inicie o jogo e divirta-se !
